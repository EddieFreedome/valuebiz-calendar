
# Valuebiz Calendar

Test fullstack tecnico per Valuebiz.




## Requirements
```Laravel 7 PHP 7.4.33.```
```Necessario NODE <= 14``` 

## Installation

Install my-project with npm

```bash
  Compile .env file

  Composer install

  npm install
  
  php artisan migrate
  
  php artisan key:generate

  php artisan serve
```
